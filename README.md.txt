# Bluestock Fintech — Mutual Fund Analytics Capstone

## Project Overview
End-to-end Mutual Fund Analytics Platform built using Python, SQL, and Power BI.
Covers ETL pipeline, EDA, performance metrics, risk analytics and interactive dashboard.

## Tech Stack
- Python, Pandas, NumPy, Matplotlib, Seaborn
- SQLite, SQLAlchemy
- Power BI Desktop
- Jupyter Notebook

## How to Run
1. Install dependencies: pip install pandas numpy matplotlib seaborn plotly sqlalchemy scipy
2. Place all 10 CSV files in data/raw/
3. Run notebooks in order from 01 to 05
4. Open dashboard in Power BI Desktop

## Key Findings
1. SBI MF dominates with Rs.12.5 lakh crore AUM
2. SIP inflows hit all-time high of Rs.31,002 crore in Dec 2025
3. Total folios doubled from 13.26 to 26.12 crore in 4 years
4. Large cap funds highly correlated
5. 26-35 age group is the largest SIP investor segment
