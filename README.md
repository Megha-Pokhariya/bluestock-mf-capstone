# Bluestock Fintech — Mutual Fund Analytics Platform

## Project Overview
End-to-end Mutual Fund Analytics Platform built using Python, SQL, and Power BI.

## Tech Stack
- Python, Pandas, NumPy, Matplotlib, Seaborn
- SQLite, SQLAlchemy
- Power BI Desktop
- Jupyter Notebook

## How to Run
1. Install: pip install pandas numpy matplotlib seaborn plotly sqlalchemy scipy
2. Run notebooks in order from 01 to 05
3. Open dashboard in Power BI Desktop

## Key Findings
1. SBI MF dominates with Rs.12.5 lakh crore AUM
2. SIP inflows hit all-time high of Rs.31,002 crore in Dec 2025
3. Total folios doubled from 13.26 to 26.12 crore
4. 26-35 age group is the largest SIP investor segment
5. Financial Services and IT account for 40% of equity portfolios
